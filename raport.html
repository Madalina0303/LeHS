Introduction

Proiectul LeHS are ca scop exersarea si verificarea cunostintelor despre Html si Css

Front End

Paginile site-ului web sunt responsive iar tematica jocului este reprezentata de o insula locuita de pinguini

Sign Up & Login

Prima pagina a aplicatiei este "index.html" prin care utilizatorul poate accesa jocul fie prin logare fie prin inregistrare. Deasemenea de pe aceasta pagina utilizatorul are acces la galeria jocului

Menu

Se pot selecta diverse pagini ale aplicatiei cum ar fi : selectarea nivelului de joc, vizualizarea unui clasament, a profilului,setari.

Levels

Din meniu se ajunge pe pagina "levels.html" unde utilizatorul poate selecta o categorie la care vrea sa-si verifice cunostintele precum si un nivel de dificultate.

Beginner-HTML

Se propun jucatorului o serie de 4 exercitii. Un exercitiu efectuat corect deblocheza urmatorul exercitiu. In cazul celui de-al 4 lea exercitiu se deblocheaza nivelul urmator.
